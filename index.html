Assignments

Assignment 1- Basic HTML

The task of Assignment 1 was to create a webpage on a topic of our choice using the basic skills of HTML we had learned so far. I chose my topic to be on Carolina Wrens, SC's state bird and a very common one here at USC. The structure for this main page was also established.

Assignment 2- Basic CSS

The task of Assignment 2 was to create a webpage on a topic of our choice using the basic skills of CSS we had learned so far. I chose my topic to be on Northern Mockingbirds, another extremely common bird that can often be found singing on pylons and buildings throughout campus. Additional styling was also applied to this main page.

Assignment 3- Page Layout

The task of Assignment 3 was to mimic a PDF of a webpage using flexbox.

Assignment 5- Recreate CSS Page

The task of Assignment 5 was to recreate the entire structure of the College of Engineering and Computing webpage ( linked here ).

Assignment 5- JavaScript, Buttons, Functions, and More

The task of Assignment 7 was to create a page where the user can click to change an image, use a range slider to rotate an image, and click a column to add stars.

Assignment 8- Conditionals

The task of Assignment 8 was to create a page where the user can choose between interacting with two exercises. Exercise 1 consists of entering commands to change to various images, while Exercise 2 consists of using a range slider to change to different yoga images.
